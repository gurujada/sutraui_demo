# Sutra UI Demo

The official documentation and demo site for [Sutra UI](https://github.com/gurujada/sutra_ui) - a modern, accessible component library for Phoenix LiveView.

## What is Sutra UI?

Sutra UI is a comprehensive UI component library built specifically for [Phoenix LiveView](https://hexdocs.pm/phoenix_live_view). It provides a collection of beautifully designed, accessible, and customizable components that help developers build modern web applications faster.

Key features:
- **40+ Components** - Buttons, forms, modals, navigation, data display, and more
- **Accessibility First** - WCAG compliant with proper ARIA attributes and keyboard navigation
- **Dark Mode** - Built-in light and dark theme support
- **Customizable** - Easily themeable with CSS variables
- **Zero JavaScript Dependencies** - Pure Phoenix LiveView with colocated JS hooks where needed
- **Copy & Paste Ready** - Use components directly or customize to fit your needs

## About This Site

This repository contains the official documentation and interactive demo site for Sutra UI. It showcases all available components with:
- Live, interactive previews
- Copy-ready code examples
- Usage guidelines and best practices
- Accessibility notes

## Local Development

To run this demo site locally:

```bash
# Clone the repository
git clone https://github.com/gurujada/sutraui_demo.git
cd sutraui_demo

# Install dependencies
mix setup

# Start the Phoenix server
mix phx.server
```

Visit [localhost:4000](http://localhost:4000) to view the site.
